git guide

`

  1. npm i -g hexo
  2. 在一个空目录 hexo init 初始化博客系统
  3. hexodir\source_posts 中 md 文档就是博客文档
  4. 撰写博客 新建 .md文件
  5. 把 md文件 转html hexo g
  6. cmd 中 hexo s 后在浏览器 localhost:4000
  7. 与 github 关联
  8. npm i hexo-deployer-git –save 插件安装
  9. 配置文件 根目录下 _config.yml
  10. `
